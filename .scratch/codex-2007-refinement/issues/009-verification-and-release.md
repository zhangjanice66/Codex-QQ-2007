# Issue 09：建立真实 Codex 验收矩阵并生成发布包

Label: `ready-for-agent`

## Parent

[Codex 2007：原生结构稳定与 QQ2007 视觉深化](../spec.md)

## What to build

扩展现有验证入口，使其在真实 Codex Desktop 会话中检查首页、项目、任务、审查、好友栏冲突、模式切换和恢复/重应用行为，并在常见缩放和较小窗口高度下验证布局。所有门禁通过后更新用户可见变更记录并生成最终发布包及校验值。

## Acceptance criteria

- [ ] 快速回归和完整 macOS 测试覆盖幂等注入、增量观察、原生属性保护、模式切换和完整清理。
- [ ] 真实会话验证覆盖首页、项目、普通任务和原生右面板，并检查原生操作按钮可见可点。
- [ ] 100%、125% 和 150% 缩放以及较小窗口高度下不存在面板覆盖、横向溢出或关键控件裁切。
- [ ] 恢复后无主题残留，重新应用后每个主题区域恰好出现一次。
- [ ] 深度仿制版、原版视图切换和右面板冲突场景均保存脱敏验收截图。
- [ ] 完整测试、doctor、live verify 和恢复/重应用 smoke 通过后才更新变更记录、版本并生成发布包与 SHA-256。
